# How to run

## Clone

```bash
git clone git@github.com:MechaKnightz/frontend-coding-challenge.git
```

## Install dependencies

```bash
cd frontend-coding-challenge
yarn
```

## Dev env

```bash
yarn start
```

## Build production

```bash
yarn build
```
